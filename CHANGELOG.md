# 📝 변경 이력

## v1.2.0 - 2026-03-08 23:19

### ✨ 새로운 기능
- **한국 시간 정보 자동 인식**
  - AI가 현재 한국 시간을 정확히 알 수 있음
  - "지금 몇 시야?", "현재 시간 알려줘" 등의 질문에 정확한 답변
  - 년, 월, 일, 요일, 시간까지 모두 표시

### 🔧 개선 사항
- OpenAI API 호출 시 시스템 프롬프트에 현재 시간 자동 추가
- 기본 모드에서도 시간 관련 질문 답변 가능
- 한국 표준시(Asia/Seoul) 적용

### 📊 작동 예시

**질문:** "지금 몇 시야?"

**AI 답변 (API 모드):**
```
현재 한국 시간은 2026년 3월 8일 토요일 오후 11시 19분입니다! 🕐
```

**기본 답변 (비API 모드):**
```
현재 시간을 알려드릴게요! 🕐

📅 2026년 3월 8일 토요일
⏰ 오후 11시 19분 (한국 시간)
```
