# 🎯 챗봇 기능 목록

## ✨ 실시간 정보 제공

### 🕐 현재 시간
```
질문: "지금 몇 시야?"
답변: 한국 시간 (Asia/Seoul) 기준 정확한 시간 제공
```

### 🌤️ 날씨 정보
```
질문: "서울 날씨 어때?"
답변: 
- 현재 기온
- 체감 온도
- 날씨 상태
- 습도
```
- **API**: wttr.in (무료)
- **지역**: 서울 기준
- **실시간 업데이트**

### 💰 암호화폐 시세
```
질문: "비트코인 가격 얼마야?"
답변:
- 비트코인(BTC) 현재가 (KRW)
- 이더리움(ETH) 현재가 (KRW)
- 24시간 변동률
```
- **API**: CoinGecko (무료)
- **화폐**: 원화(KRW) 기준
- **실시간 업데이트**

## 🎯 사용 예시

### 1. 실시간 정보 조회
```
👤 "서울 날씨 알려줘"
🤖 서울 현재 날씨 정보입니다! 🌤️

   🌡️ 기온: 5°C (체감 2°C)
   ☁️ 날씨: 구름 조금
   💧 습도: 45%
   
   아파트 견학하기 좋은 날씨네요!
```

### 2. 코인 시세 확인
```
👤 "비트코인 가격"
🤖 암호화폐 현재 시세입니다! 💰

   ₿ 비트코인(BTC)
   가격: 85,430,000원
   24시간: +2.45% 📈
   
   ⓔ 이더리움(ETH)
   가격: 4,520,000원
   24시간: -1.23% 📉
```

### 3. 일반 지식 질문
```
👤 "2차 방정식이 뭐야?"
🤖 2차 방정식은 ax² + bx + c = 0 형태의 
   방정식입니다. 여기서 a ≠ 0이고...
   [자세한 설명 계속]
```

### 4. 아파트 상담
```
👤 "분양가 얼마예요?"
🤖 분양가는 아파트별로 상이합니다. 😊
   
   현재 분양 중인 아파트:
   • 힐스테이트 달성공원역
   • 범어 자이르네
   ...
```
